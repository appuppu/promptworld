# Prompt World

A black & white minimal physics sandbox platform — hacking together the
system structure of Mario Maker and the UGC explosion of Roblox, where
players use their own AI (Claude) to invent custom physics rules and
build stages.

## Core Concept

- **Stages are data, not code.** Every stage deploys as a lightweight JSON
  text document: time limit, goal conditions, block layout, and
  trigger-and-action physics rules. The game client is an interpreter.
- **Create by prompting, play anywhere.** Users create stages by talking to
  their own Claude (via the Prompt World MCP server); stages are shared as
  URLs that open the WebGL player in any browser, or deep-link into the
  mobile app. The stage JSON is fetched from the stage server instantly.
- **Creator clear-check.** A stage can only be deployed if its creator
  beats it within the time limit — a proven quality filter.
- **AI-native creation.** Because stages are pure text, an AI can generate,
  validate, and remix them directly from natural-language prompts. That is
  the "Prompt" in Prompt World.
- **Future:** local P2P stage sharing between nearby players (StreetPass-style).

## Infrastructure (planned — optimized for $0)

All-in on Cloudflare's free tier:

- **Cloudflare Pages** — hosts the WebGL player build (free, unlimited bandwidth)
- **Cloudflare Workers** — stage API + the remote MCP server (free: 100k requests/day)
- **Cloudflare D1** — stage JSON + clear-certificate storage (free: 5 GB, 5M reads/day)

Stages are a few KB of JSON, so these limits are effectively unreachable at
launch scale. The only future paid items are a custom domain (~$10/yr, needed
for mobile Universal Links) and app store developer accounts.
